# Mind in PPT - 演示风格系统设计

> **版本**: v2.0
> **设计目标**: 提供覆盖全职场场景的演示风格系统

---

## 一、风格分类体系

### 1.1 风格分类原则

**按照"使用场景"分类**，而非视觉风格

**涵盖 6 大类职场场景**:
1. 📊 **商务汇报** - 正式商务场合
2. 💼 **职场办公** - 日常工作汇报
3. 🎤 **行业交流** - 行业会议/研讨会
4. 💡 **创意脑暴** - 创意讨论/头脑风暴
5. 📚 **学术教育** - 培训/教学场景
6. 🚀 **产品发布** - 产品/项目发布会

---

## 二、完整风格列表

### 📊 商务汇报类 (3种)

#### 2.1 高管汇报
- **适用场景**: 向高管层汇报、年度总结、战略规划
- **视觉特点**: 深蓝+金色，简洁专业，突出数据
- **语气**: 正式、精炼、结果导向
- **图标推荐**: 数据、图表、奖杯、靶心

#### 2.2 客户提案
- **适用场景**: 向客户提案、商务谈判
- **视觉特点**: 温暖色调（橙+蓝），友好专业
- **语气**: 亲和、自信、客户导向
- **图标推荐**: 握手、对话、合同、笑脸

#### 2.3 投融资路演
- **适用场景**: 融资路演、投资人会议
- **视觉特点**: 深色背景+荧光色，科技感
- **语气**: 激动人心、愿景驱动
- **图标推荐**: 火箭、上升曲线、地球、星星

---

### 💼 职场办公类 (3种)

#### 2.4 周报月报
- **适用场景**: 定期工作汇报
- **视觉特点**: 清爽蓝绿，简洁明了
- **语气**: 客观、条理清晰
- **图标推荐**: 日历、清单、完成、进度

#### 2.5 项目进展
- **适用场景**: 项目里程碑汇报
- **视觉特点**: 甘特图风格，时间轴
- **语气**: 严谨、数据驱动
- **图标推荐**: 时间轴、里程碑、流程图、检查点

#### 2.6 团队总结
- **适用场景**: 团队季度/年度总结
- **视觉特点**: 温暖橙色，突出团队
- **语气**: 鼓励、感恩、展望未来
- **图标推荐**: 团队、握手、鼓掌、目标

---

### 🎤 行业交流类 (3种)

#### 2.7 行业研讨会
- **适用场景**: 行业会议、专业论坛
- **视觉特点**: 深灰+强调色，专业学术
- **语气**: 权威、专业、洞察
- **图标推荐**: 图表、分析、趋势、洞察

#### 2.8 经验分享
- **适用场景**: 内部经验分享会
- **视觉特点**: 清新绿色，成长感
- **语气**: 谦虚、分享、实用
- **图标推荐**: 灯泡、笔记、分享、成长

#### 2.9 行业趋势
- **适用场景**: 趋势解读、行业前瞻
- **视觉特点**: 科技感+数据可视化
- **语气**: 前瞻、分析、引导
- **图标推荐**: 趋势、未来、变化、洞察

---

### 💡 创意脑暴类 (2种)

#### 2.10 创意风暴
- **适用场景**: 头脑风暴、创意讨论
- **视觉特点**: 鲜艳撞色、活泼有趣
- **语气**: 轻松、开放、激发灵感
- **图标推荐**: 灯泡、闪电、星星、彩虹

#### 2.11 方案共创
- **适用场景**: 团队共创、工作坊
- **视觉特点**: 暖色调、协作感
- **语气**: 参与式、共同创造
- **图标推荐**: 团队、对话、协作、创意

---

### 📚 学术教育类 (2种)

#### 2.12 培训教学
- **适用场景**: 内部培训、知识分享
- **视觉特点**: 清晰结构、教学感
- **语气**: 循序渐进、耐心细致
- **图标推荐**: 书籍、黑板、学习、笔记

#### 2.13 学术报告
- **适用场景**: 学术报告、论文答辩
- **视觉特点**: 灰色系、严谨规范
- **语气**: 严谨、客观、学术化
- **图标推荐**: 引用、数据、研究、论文

---

### 🚀 产品发布类 (2种)

#### 2.14 产品发布会
- **适用场景**: 新品发布
- **视觉特点**: 高对比、视觉冲击力
- **语气**: 激动、震撼、亮点突出
- **图标推荐**: 星星、火箭、新品、亮点

#### 2.15 技术分享
- **适用场景**: 技术分享、架构评审
- **视觉特点**: 代码风格、极客感
- **语气**: 技术化、深度、专业
- **图标推荐**: 代码、芯片、终端、API

1. **点击下拉菜单** → 展开 6 大类

3. **点击具体风格** → 选中并收起菜单
4. **显示当前选择** → 显示风格名称和适用场景

## 四、技术实现

### 4.1 风格数据结构

```typescript
interface StyleOption {
  id: string;              // 唯一标识
  category: StyleCategory;  // 所属类别
  name: string;            // 风格名称
  icon: string;            // 图标
  description: string;     // 适用场景描述

  // 配色方案
  colors: StyleColors;

  // AI 提示词
  aiPrompt: {
    tone: string;
    structure: string;
    example: string;
  };
}

type StyleCategory =
  | 'business'       // 商务汇报类
  | 'office'         // 职场办公类
  | 'conference'     // 行业交流类
  | 'creative'       // 创意脑暴类
  | 'education'      // 学术教育类
  | 'product';       // 产品发布类
```

// 使用 Headless UI 的 Select 组件
